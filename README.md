# Tools
一些簡單的小工具
1. add_romaji.py: 日/中歌詞整理與加上拼音，需要cutlet
2. AudioFadeTool.py: 音量漸弱工具，需要pycaw和comtypes
3. bilimix.py: 合併BiliBili下載的影片，有ts和blv兩種模式
4. colorprint.py: Windows版套色輸出
5. cut_sub.py: srt字幕檔的擷取與時間平移
6. do_ffmpeg.py: 批量處理檔案，有m4a轉檔為mp3、提取mp4影片音檔為m4a兩種功能，變數DO修改當前功能
7. mangaowl.py: 陽春的下載器，內有說明，需要requests和beautifulsoup
8. manhuaren.py: 陽春的下載器，內有說明，需要selenium
9. ScanPDFIncrementalChange.py: 壓縮pdf編輯器的增量儲存，原理是掃描檔案末端的EOF數量，需要pikepdf
10. tera_rename.py: 因為terabox不能上傳影片而做的，指定資料夾首刪掉內部檔案副檔名最後一字
